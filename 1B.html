(Anchor Tag): Used to create hyperlinks to other web pages, email addresses, or files. href (Hypertext REFerence): Specifies the URL or link destination. target Attribute: Specifies where to open the linked document. _blank opens the link in a new tab. _self (default) opens in the same tab. (Image Tag): Used to embed images in a web page. src defines the image path. alt provides alternative text if the image doesn't load. width sets the image width in pixels. Code: Hyperlink Example
